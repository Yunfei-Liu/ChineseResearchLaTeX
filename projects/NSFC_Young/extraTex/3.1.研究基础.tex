\subsubsection{相关研究工作积累与已取得的成绩}

% 申请人长期深耕于\textbf{3D计算机视觉、生成式人工智能与多模态人机交互}领域，在CVPR, ICCV, ICLR, AAAI, TPAMI, TVCG等顶级会议期刊发表多篇高水平论文。申请人团队在``数字空间''构建的高保真感知与生成技术体系，不仅在算法层面为本项目的Social-MLA模型提供了核心模块（感知编码器与流匹配解码器），更在数据层面为解决具身智能``高表现力数据匮乏''难题奠定了坚实基础。

申请人长期深耕于\textbf{多模态交互、3D数字人高保真重构与具身智能生成}的交叉领域。申请人博士毕业于北京航空航天大学虚拟现实技术与系统全国重点实验室（师从陆峰教授），在博士期间系统性地开展了基于物理的人工智能与视线感知研究，深刻理解并深度参与了“共身智能（Cobodied AI）”前沿理论体系的探索与构建。随后，申请人进入粤港澳大湾区数字经济研究院（IDEA）担任高级算法研究员，全面主导了以人为核心的AIGC多模态内容生成与前沿系统落地。

这一“从国家重点实验室的基础理论探索，到顶尖新型研发机构的工程落地”的复合科研经历，使得申请人兼具了宏观的科学视野与扎实的底层代码架构能力。近年来，申请人围绕本项目的核心科学问题，CVPR, ICCV, ICLR, TPAMI, TVCG 等人工智能、模式识别与计算机视觉顶级会议和期刊发表多篇前沿学术论文，在产学研深度融合与前沿技术转化方面取得了突出的成绩。前期在“数字物理空间”积累的感知与生成技术，以及在真机部署上的工程实践，为本项目提供了从底层算法到系统落地的全栈式支撑。具体成绩归纳为以下三个维度：


\paragraph{\textbf{1. 高维多模态感知与Analysis-by-Synthesis重构（支撑模块一：数据解析层）。}}
本项目第一层需解决复杂遮挡下“表情-视线-动作”的同步解耦。申请人在该领域已具备深厚的算法储备：
\begin{itemize}
    \item \textbf{基于Analysis-by-Synthesis的高精度人体几何重构：} 申请人提出了 PEAR （2026）~\cite{wu2026pear}框架，攻克了像素级对齐的全身高表现力网格恢复难题；同时，最近发表于\textbf{ICCV 2025}的 GUAVA~\cite{GUAVA} 工作中实现了Zero-shot的高质量基于3DGS的人体重建与驱动生成。这些工作为本项目从互联网视频中反向提取 Embodied-Social-100k 数据集提供了现成、可靠的算法基座。
    \item \textbf{微表情与视线精细化建模：} 在表情解析方面，最新发表于 \textbf{ICLR 2025} 的TEASER~\cite{liu2025TEASER}工作中，申请人提出了基于Token增强的空间建模方法，实现了细致微表情3D人头重建；在视线方向估计方面，申请人发表于 \textbf{ICCV 2021} 的工作利用离群点引导的协同自适应机制~\cite{liu2021PnP_GA}，大大提升了视线估计的泛化性。这解决了本项目感知阶段“细粒度情感特征（微表情+眼神）捕捉”的技术瓶颈。
\end{itemize}

\paragraph{\textbf{2. 跨模态语音驱动与多部位协同生成模型（支撑模块二：Social-MLA认知生成层）}。}
针对本项目核心架构Social-MLA所需的“离散语义规划+连续流形解码”能力，申请人已在语音驱动全身动作生成领域取得多项标志性成果：
\begin{itemize}
    \item \textbf{全身协同生成：} 在DiffSHEG~\cite{Chen2024DiffSHEG}工作中，申请人提出了一种基于扩散模型的实时语音驱动全身3D表情与手势协同生成框架，从底层数学架构上验证了利用连续概率模型生成高平滑度具身动作的可行性。
    \item \textbf{对话级多模态交互：} 申请人最新的MANGO（2026）~\cite{Zhu2026MANGO}工作突破了自然多说话人3D对话生成的难题；而在 \textbf{ICCV 2023} 发表的MODA工作中~\cite{Liu2023MODA}，通过双重注意力机制（Dual Attentions）实现了高质量的音频驱动人像动画。这些在时序逻辑与对话轮换机制上的研究，为本项目多模交互Token预测（MITP）提供了成熟的特征融合与自回归网络设计经验。
\end{itemize}

\paragraph{\textbf{3. Sim-to-Real物理仿真与真机控制系统集成（支撑模块三：物理闭环层）。}}
申请人不仅具备扎实的算法理论功底，而且拥有丰富的具身智能仿真、控制与端到端系统架构开发经验：
\begin{itemize}
    \item \textbf{运动重定向与物理仿真：} 申请人已成功实现了基于GMR（通用动作重定向）的人形机器人Motion Retargeting算法。该算法打通了从SMPL-X人体参数到多种复杂机器人URDF的端到端IK（逆运动学）学习，并已在MuJoCo与Isaac Lab物理引擎下完成了深度仿真验证。
    \item \textbf{底层控制与实时系统架构：} 申请人早期曾基于PID控制实现了机械臂的稳定抓取与控制调参，具备扎实的控制论基础。此外，申请人主导研发了“高保真实时交互数字人系统”，打通了ASR、LLM、TTS与Audio-to-Motion的全栈链路，在消费级显卡上实现了毫秒级的端到端响应。这一系统的高并发调度与显存优化经验，将有力保障Social-MLA大模型在物理机器人机载计算平台上的实时性与鲁棒性。
\end{itemize}

\subsubsection{研究风险及应对措施}

{\textbf{风险 1：虚实鸿沟（Sim-to-Real Gap）导致的底盘动态失稳与高频抖动。}}

\textbf{描述：} 虽然在Isaac Lab中能实现动力学收敛，但由于真实环境摩擦力分布不均、减速器背隙与伺服延迟等非线性物理约束，生成的表现力动作可能在真机部署时引发底盘晃动或末端抖动。\\
\textbf{应对措施：} （1）在RL追踪策略训练阶段，大规模引入域随机化技术，针对目标机器人的连杆质量、阻尼与控制延迟进行高斯噪声注入，强制策略网络学习鲁棒的代偿机制；（2）依托星辰智能（Astribot）硬件团队在全身全身动力学控制（WBC）方面的底层接口支持，将本项目的RL高层指令与底层的低延迟力矩补偿算法进行融合过滤。

{\textbf{风险 2：Social-MLA 级联架构在机载设备上的推理延迟问题。}}

\textbf{描述：} 人机共身交互对延迟极其敏感。本项目采用的“LM自回归预测 + 流匹配ODE求解”的双层架构，具有较高的计算复杂度，在机器人边缘算力（如机载Orin芯片）上可能面临推理瓶颈。\\
\textbf{应对措施：} （1）利用申请人在“实时交互数字人系统”中积累的显存优化与流水线并行技术，实现音频特征提取与Token自回归的异步并发；（2）采用最新的ODE加速求解器（如DPM-Solver或一致性模型 Consistency Models），将流匹配的采样步数从传统的百步级压缩至个位数（4-8 Steps），在极低牺牲动作保真度的前提下，确保端到端动作生成的毫秒级实时响应。
